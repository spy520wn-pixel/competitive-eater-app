# 大胃王APP 需求设计方案与技术设计方案

---

## 第一部分：需求设计方案

### 1. 项目概述

#### 1.1 项目背景
大胃王APP是一款面向美食爱好者和大胃王挑战者的社交应用，用户可以记录自己在各类餐厅（尤其是自助餐）的就餐战绩，与其他用户进行排名比拼，并通过AI生成精美的战绩分享图片。

#### 1.2 目标用户
- 自助餐爱好者
- 大胃王挑战参与者
- 美食社交用户
- 餐厅老板（可选，用于数据统计）

#### 1.3 核心价值
- **记录**：量化就餐成果，让每次美食体验都有据可查
- **竞技**：通过排名系统激发用户的竞争意识
- **社交**：通过分享功能扩大社交影响力
- **趣味**：AI生成的创意图片增加分享乐趣

- 个人战绩统计（总次数、总消费、最佳战绩）
- 历史战绩列表

#### 2.2 就餐记录模块

##### 2.2.1 选择店铺
- 搜索店铺（支持关键词、位置）

- **基础信息**：
  - 就餐日期时间
  - 就餐人数
  - 消费金额

- **菜品记录**（核心功能）：
  - 选择菜品分类（肉类、海鲜、主食、甜点、饮料等）

  - 自定义菜品名称

  - 系统根据菜品数量自动计算"战斗力"

  - 备注栏（记录特殊战绩，如"首次挑战成功"）

- 完整的菜品清单
- 战绩评分和排名

#### 4.2 排名计算逻辑

```java
/**
 * 战斗力评分算法
 * 基础分 + 菜品加成 + 种类加成 + 时长加成
 */
public int calculateScore(Record record, List<RecordItem> items) {
    int baseScore = 100;

    // 菜品数量加成（每道菜+10分）
    int itemBonus = items.size() * 10;

    // 总数量加成（每单位+5分）
    int quantityBonus = items.stream()
        .mapToInt(item -> item.getQuantity().intValue() * 5)
        .sum();

    // 种类丰富度加成（每种分类+20分）
    long categoryCount = items.stream()
        .map(RecordItem::getCategory)
        .distinct()
        .count();
    int categoryBonus = (int)categoryCount * 20;

    // 时长加成（吃的越快加分越多，30分钟内+50分）
    int durationBonus = record.getDuration() <= 30 ? 50 : 0;

    return baseScore + itemBonus + quantityBonus + categoryBonus + durationBonus;
}
```

### 5. 关键技术实现

#### 5.1 排名系统实现（Redis Sorted Set）

```java
@Service
public class RankService {

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    private static final String RANK_KEY_PREFIX = "rank:";

    /**
     * 更新用户排名
     */
    public void updateRank(Long userId, Long shopId, int score) {
        // 店铺排名
        String shopRankKey = RANK_KEY_PREFIX + "shop:" + shopId;
        redisTemplate.opsForZSet().add(shopRankKey, userId, score);

        // 全局排名
        String globalRankKey = RANK_KEY_PREFIX + "global";
        redisTemplate.opsForZSet().add(globalRankKey, userId, score);
    }

    /**
     * 获取店铺排名（Top N）
     */
    public List<RankVO> getShopRank(Long shopId, int topN) {
        String shopRankKey = RANK_KEY_PREFIX + "shop:" + shopId;
        Set<ZSetOperations.TypedTuple<Object>> ranks =
            redisTemplate.opsForZSet().reverseRangeWithScores(shopRankKey, 0, topN - 1);

        return ranks.stream().map(tuple -> {
            RankVO vo = new RankVO();
            vo.setUserId(Long.parseLong(tuple.getValue().toString()));
            vo.setScore(tuple.getScore().intValue());
            return vo;
        }).collect(Collectors.toList());
    }

    /**
     * 获取用户排名
     */
    public Long getUserRank(Long userId, Long shopId) {
        String shopRankKey = RANK_KEY_PREFIX + "shop:" + shopId;
        return redisTemplate.opsForZSet().reverseRank(shopRankKey, userId);
    }
}
```

#### 5.2 AI图片生成实现

```java
@Service
public class AIShareService {

    @Value("${ai.api.key}")
    private String apiKey;

    @Value("${ai.api.url}")
    private String apiUrl;

    /**
     * 生成战绩分享图片
     */
    public String generateShareImage(Record record, String style) {
        // 构建提示词
        String prompt = buildPrompt(record, style);

        // 调用AI API
        Map<String, Object> request = new HashMap<>();
        request.put("prompt", prompt);
        request.put("size", "1024x1024");
        request.put("style", style);

        HttpHeaders headers = new HttpHeaders();
        headers.set("Authorization", "Bearer " + apiKey);
        headers.setContentType(MediaType.APPLICATION_JSON);

        HttpEntity<Map<String, Object>> entity = new HttpEntity<>(request, headers);
        ResponseEntity<Map> response = restTemplate.exchange(
            apiUrl, HttpMethod.POST, entity, Map.class);

        // 获取图片URL
        String imageUrl = (String) response.getBody().get("url");

        // 下载并保存到OSS
        return uploadToOSS(imageUrl);
    }

    /**
     * 构建AI提示词
     */
    private String buildPrompt(Record record, String style) {
        StringBuilder prompt = new StringBuilder();
        prompt.append("生成一张大胃王战绩海报，风格：").append(style).append("。");
```

```java
@Service

        g2d.fillRect(0, 0, width, height);

        // 设置字体
        Font titleFont = new Font("微软雅黑", Font.BOLD, 18);
        Font normalFont = new Font("微软雅黑", Font.PLAIN, 14);
        Font smallFont = new Font("微软雅黑", Font.PLAIN, 12);

        // 绘制标题
        g2d.setFont(titleFont);
        g2d.setColor(Color.BLACK);

        // 绘制分割线
        drawDashedLine(g2d, 20, y, width - 20, y);
        y += 20;

        // 绘制店铺信息
        g2d.setFont(normalFont);
        g2d.drawString("店铺：" + record.getShopName(), 20, y);
        y += 25;
        g2d.drawString("日期：" + record.getEatDate(), 20, y);
        y += 25;
        g2d.drawString("人数：" + record.getPeopleCount() + "人", 20, y);
        y += 30;

        // 绘制分割线
        drawDashedLine(g2d, 20, y, width - 20, y);
        y += 20;

        // 绘制菜品列表
        g2d.setFont(smallFont);
        g2d.drawString("菜品", 20, y);
        g2d.drawString("数量", 280, y);
        y += 25;

        for (RecordItem item : record.getItems()) {
            g2d.drawString(item.getDishName(), 20, y);
            g2d.drawString(item.getQuantity() + item.getUnit(), 280, y);
            y += 20;
        }

        // 绘制分割线
        y += 10;
        drawDashedLine(g2d, 20, y, width - 20, y);
        y += 20;

        // 绘制统计信息
        g2d.setFont(normalFont);
        g2d.drawString("总计菜品：" + record.getItems().size() + "道", 20, y);
        y += 25;
        g2d.drawString("战斗力评分：" + record.getScore() + "分", 20, y);
        y += 25;
        g2d.drawString("消费金额：¥" + record.getTotalAmount(), 20, y);

    private void drawCenteredString(Graphics2D g2d, String text, int width, int y) {
        FontMetrics fm = g2d.getFontMetrics();
        int x = (width - fm.stringWidth(text)) / 2;
        g2d.drawString(text, x, y);
    }

    private void drawDashedLine(Graphics2D g2d, int x1, int y1, int x2, int y2) {
        g2d.setStroke(new BasicStroke(1, BasicStroke.CAP_BUTT, BasicStroke.JOIN_BEVEL, 0, new float[]{5}, 0));
        g2d.drawLine(x1, y1, x2, y2);
    }
}
```
